# Помощник координатора встреч

Это веб-приложение для помощи координаторам в планировании и проведении встреч. Оно предоставляет интерактивные блоки с описаниями различных действий, которые можно выполнять во время встречи.

## Функционал

- **Интерактивные блоки**: Кликните на блок, чтобы выделить его (становится фиолетовым). Повторный клик снимает выделение (белый с фиолетовой границей).
- **Последовательное отображение описаний**: При выделении нескольких блоков их описания показываются последовательно в правой панели.
- **Действия**:
  - Синхронизироваться
  - Утвердить план
  - Мозговой штурм
  - Демонстрация
  - Брифинг
  - Распределить задачи
  - Оценка
  - Декомпозиция задач

## Как использовать

1. Откройте файл `site.html` в любом современном веб-браузере.
2. Кликайте на блоки слева, чтобы выделить нужные действия.
3. Читайте описания в правой панели.
